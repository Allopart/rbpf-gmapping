trscale

Homogeneous transformation for pure scale

T = trscale(s) is a homogeneous transform (4x4) corresponding to a pure scale change. If s is a scalar the same scale factor is used for x,y,z, else it can be a 3-vector specifying scale in the x-, y- and z-directions.
